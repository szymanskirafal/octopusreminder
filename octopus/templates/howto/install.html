How to add web app to the Home screen of your iPhone.

Tap Safari icon.

Write octopusreminder.com in the search bar and click Go button.

Tap the share icon.

Tap the Add to Home Screen icon.

Here you can change the name to be displayed below the icon, leave it unchanged or delete the name to have just icon without any name below it. Then you should click Add button.

The icon should apear on the screen of your iPhone.

You can move the icon between screens. Just tap and hold the icon for few seconds. It should start shaking. Then you can move it. Tap the Home Button to set it up.
